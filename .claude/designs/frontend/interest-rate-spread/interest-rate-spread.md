# 금리 스프레드 패널

## 배경

금리 탭(INTEREST_RATE)에서 이미 시장금리 + 여수신금리 지표를 함께 로드하고 있음.
이 데이터로 6개 금리 스프레드를 계산하여 카드 대시보드와 테이블 사이에 패널로 표시.

## 스프레드 정의

| # | 이름 | 계산 | 의미 |
|---|---|---|---|
| 1 | 장단기 금리차 | 국고채5년 - CD91일 | 시장 구조 판단 (핵심) |
| 2 | 중기-단기 금리차 | 국고채3년 - CD91일 | 금리 기대 방향 |
| 3 | 장기 금리 기울기 | 국고채5년 - 국고채3년 | 장기 기대 (인플레/성장) |
| 4 | 신용 스프레드 | 회사채(AA-) - 국고채3년 | 시장 리스크 수준 |
| 5 | 예대금리차 | 대출금리 - 예금금리 | 금융 부담 / 은행 구조 |
| 6 | 단기 vs 기준금리 | 콜금리 - 기준금리 | 유동성 상태 |

## 지표 매칭 (keystatName 기준)

- `국고채수익률(5년)`, `국고채수익률(3년)`, `CD수익률(91일)`
- `회사채수익률(3년,AA-)`, `예금은행 대출금리`, `예금은행 수신금리`
- `콜금리(익일물)`, `한국은행 기준금리`

## 구현 방식

- **프론트엔드만 변경**, 백엔드 수정 없음
- `ecos.js`에 스프레드 계산 함수 추가
- `index.html`에 금리 탭 선택 시 카드뷰 아래에 스프레드 패널 표시
- 카테고리가 `INTEREST_RATE`일 때만 노출

## UI 구성

카드뷰 아래, 테이블 위에 스프레드 패널 배치:
- 3x2 그리드 카드 레이아웃
- 각 카드: 스프레드명, 계산값(%p), 의미 한 줄 설명
- 양수/음수에 따른 색상 구분
