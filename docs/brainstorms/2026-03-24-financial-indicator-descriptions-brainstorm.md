# Brainstorm: 재무제표 지표별 설명 툴팁 추가

**Date**: 2026-03-24
**Status**: Draft

## What We're Building

포트폴리오 재무상세 패널의 **전체 메뉴**(재무계정, 재무지표, 배당정보, 전체재무제표, 주식수량, 소송현황, 사모자금사용, 공모자금사용)에 각 항목별 설명을 **호버 툴팁**으로 추가한다.

- 항목명 옆에 ℹ️ 아이콘 표시
- 마우스 호버 시 초보자 눈높이의 설명 표시
- 경제지표 대시보드에 이미 구현된 툴팁 패턴과 동일한 UI 사용

## Why This Approach

1. **일관성**: 경제지표 대시보드에서 이미 검증된 ℹ️ 호버 툴팁 패턴을 재사용
2. **빠른 구현**: 백엔드 API 변경 없이 프론트엔드 JS 메타데이터만 추가
3. **초보자 친화**: 전문 용어 없이 쉬운 설명으로 투자 초보자도 이해 가능
4. **비침투적**: 기존 테이블 레이아웃을 크게 변경하지 않고 아이콘만 추가

## Key Decisions

| 결정 사항 | 선택 | 이유 |
|-----------|------|------|
| 설명 범위 | 전체 메뉴 (8개 탭) | 어느 메뉴든 항목명만으로는 의미 파악이 어려움 |
| UI 방식 | ℹ️ 호버 툴팁 | 경제지표 대시보드와 동일한 패턴으로 일관성 유지 |
| 데이터 위치 | 프론트엔드 JS 메타데이터 | 백엔드 수정 불필요, 빠른 구현 |
| 설명 수준 | 초보자 눈높이 | 경제지표 설명과 동일한 스타일 유지 |
| 메타데이터 위치 | app.js 내 객체 | 경제지표 메타데이터와 동일한 위치, 일관성 |

## Scope

### 포함
- 재무계정: 매출액, 영업이익, 당기순이익, 자산총계, 부채총계, 자본총계 등
- 재무지표: PER, PBR, ROE, ROA 등 수익성/안정성/성장성/활동성 지표
- 배당정보: 주당배당금, 배당수익률, 배당성향
- 전체재무제표: 손익계산서, 재무상태표 항목
- 주식수량, 소송현황, 사모/공모자금사용 항목

### 미포함
- 백엔드 API 변경
- 새로운 컴포넌트나 라이브러리 추가
- 설명의 다국어 지원
