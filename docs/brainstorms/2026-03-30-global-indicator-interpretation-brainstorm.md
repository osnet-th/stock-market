# Brainstorm: 글로벌 경제지표 해설 추가

**Date:** 2026-03-30
**Status:** Draft

## What We're Building

글로벌 경제지표 페이지에 **각 지표별 해설을 카드/행 내에 상시 표시**하여, 숫자만으로는 판단하기 어려운 경제 방향성을 즉시 파악할 수 있게 한다.

### 핵심 문제

- 현재 글로벌 지표는 국가별 숫자만 나열되어 있어, 그 값이 좋은지 나쁜지 판단하기 어려움
- "GDP 성장률 3.2%"가 높은 건지 낮은 건지, "실업률 4.1%"가 양호한 건지 알 수 없음
- 국내 ECOS 스프레드에는 상태 판단 기준(안정/정상/주의/위험)이 있지만, 글로벌 지표에는 없음

### 해결 방향

- **레이아웃/네비게이션 구조는 현행 유지** (카테고리 → 지표 선택 → 국가 테이블)
- 각 지표에 **"높으면/낮으면 무엇을 의미하는지"** 해설 텍스트를 상시 표시
- 지표 헤더 영역에 작은 글씨로 항상 보이도록

## Why This Approach

1. **구조 변경 최소화**: 기존 카테고리/지표 선택 UX는 유지. 백엔드 변경 불필요
2. **즉각적 가치**: 해설만 추가해도 숫자 해석이 훨씬 쉬워짐
3. **ECOS와 일관성**: 국내 지표의 description 패턴과 동일한 접근

## Key Decisions

1. **해설 표시 방식**: 카드 내 상시 표시 (툴팁 아님)
   - 지표 헤더 아래에 작은 회색 텍스트로 1~2줄 해설
   - 예: "높을수록 경기 확장 신호. 마이너스면 경기 침체"

2. **해설 데이터 위치**: 프론트엔드에 정적 매핑 (32개 지표)
   - GlobalEconomicIndicatorType enum에 description 필드 추가하는 방법도 있으나, 프론트엔드만 변경하는 것이 범위가 적음
   - 향후 백엔드로 이동 가능

3. **레이아웃 유지**: 카테고리 → 지표 → 국가 테이블 구조 그대로
   - 전체 스냅샷 뷰는 추후 별도 기능으로 검토 가능

## Scope

### 포함

- 32개 글로벌 지표에 대한 해설 텍스트 정의
- 지표 헤더 영역에 해설 상시 표시 UI

### 미포함

- 상태 뱃지/게이지 바 (글로벌 지표는 국가마다 기준이 달라 일률적 범위 적용 어려움)
- 전체 카테고리 스냅샷 뷰 (별도 기능으로 분리)
- 백엔드 변경
