# 현금성 자산 이자 계산 기능

**Date:** 2026-03-25
**Status:** Brainstorm
**GitHub Issue:** #12

## What We're Building

현금성 자산(예금, 적금, CMA)에 대해 이자를 계산하고 평가금액에 반영하는 기능.

현재 CASH 자산은 `investedAmount`만 저장하고 예금/적금/CMA 구분이 없으며, 이자율이나 만기일 정보가 없음. 평가금액도 항상 원금 그대로 표시됨.

### 핵심 요구사항

1. **현금성 자산 세부 유형 구분**: 예금, 적금, CMA 3종류 지원
2. **이자 계산**: 유형별 이자 계산 후 개별 항목에 참고용으로 표시
3. **과세 유형 지원**: 일반과세(15.4%), 비과세(0%), 세금우대(9.5%) 선택
4. **평가금액 이원화**:
   - **개별 항목 표시**: 원금 + 만기 예상 이자(세후) = 예상 수령액 (참고용)
   - **포트폴리오 총 평가금액**: 입금한 원금(investedAmount) 그대로 합산 (이자 미포함)

## Why This Approach

### 유형별 이자 계산 방식

| 유형 | 입력 필드 | 이자 계산 | 평가금액 |
|------|----------|----------|---------|
| **예금** | 원금, 이자율, 만기일 | 원금 x 이자율 x (기간/12) | 항목: 원금 + 세후 이자 표시 / 총합: 원금만 |
| **적금** | 총 납입액, 이자율, 만기일 | 총액 기반 단순 계산 | 항목: 총 납입액 + 세후 이자 표시 / 총합: 납입액만 |
| **CMA** | 잔액, 이자율 | 계산 안 함 | 잔액 그대로 (이자율 표시만) |

### 과세 유형별 세율

| 과세 유형 | 세율 | 설명 |
|----------|------|------|
| 일반과세 | 15.4% (소득세 14% + 지방소득세 1.4%) | 기본값 |
| 비과세 | 0% | 비과세 상품 |
| 세금우대 | 9.5% (농특세 1.4% + 소득세 등) | 세금우대 저축 |

### 적금 계산 단순화 결정

적금은 원래 매월 불입액 기반으로 회차별 이자를 계산해야 정확하지만, 사용자가 **총 납입액 입력 방식**을 선택함. 예금과 동일하게 총액 기반으로 단순 계산. 정확도는 낮지만 입력이 간편함.

### CMA 평가금액 미반영 결정

CMA는 수시입출금이라 만기 개념이 없어 이자 예측이 어려움. 이자율은 참고용으로 표시만 하고, 평가금액에는 잔액 그대로 유지.

## Key Decisions

1. **세부 유형**: 예금/적금/CMA 3종류 (enum으로 관리)
2. **과세 유형**: 일반과세/비과세/세금우대 3종류 (enum으로 관리)
3. **적금 계산**: 총 납입액 기반 단순 계산 (월 불입액 입력 아님)
4. **CMA**: 이자율 표시만, 평가금액 변동 없음
5. **포트폴리오 총 평가금액에는 원금(입금액)만 합산** — 이자는 개별 항목에 참고 정보로만 표시
6. **이자 계산은 프론트엔드에서 수행** (기존 주식 평가금액 계산과 동일한 패턴)

## Resolved Questions

- 적금 월 불입액 vs 총액 입력 -> **총액 입력** (단순화)
- CMA 이자 평가금액 반영 여부 -> **미반영** (이자율 표시만)
- 과세 유형 범위 -> **일반/비과세/세금우대 3가지**

## 영향 범위 (예상)

### 백엔드
- `CashItemEntity`에 필드 추가: `cashType`, `interestRate`, `maturityDate`, `taxType`
- `CashDetail` 도메인 모델 신규 생성
- `PortfolioItem`에 `CashDetail` 조합 추가
- 관련 DTO, Mapper 수정
- Flyway 마이그레이션

### 프론트엔드
- 현금 자산 등록/수정 폼에 세부 유형, 이자율, 만기일, 과세 유형 입력 필드 추가
- `getEvalAmount()`는 CASH 유형에 대해 기존대로 원금 반환 (총 평가금액용)
- 별도로 예상 수령액(원금+이자) 계산 함수 추가 (개별 항목 참고 표시용)
- 현금 자산 목록에 이자율, 만기일, 예상 수령액 표시
