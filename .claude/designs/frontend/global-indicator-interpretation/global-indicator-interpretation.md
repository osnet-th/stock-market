# 글로벌 경제지표 해설 추가

## 배경

글로벌 경제지표 페이지에서 국가별 숫자만 나열되어 있어, 해당 값이 경기 확장/침체 중 어떤 신호인지 판단하기 어려움. 각 지표에 해설을 상시 표시하여 즉각적인 판단이 가능하도록 개선.

## 핵심 결정

- **프론트엔드만 변경** (백엔드 변경 없음)
- 지표 헤더 영역에 해설 텍스트를 항상 표시 (툴팁 아님)
- `global.js`에 32개 지표별 해설 매핑 객체 추가

## 구현

### 1. 해설 매핑 (global.js에 추가)

```javascript
const GLOBAL_INDICATOR_DESCRIPTIONS = {
    // 무역/GDP
    'BALANCE_OF_TRADE': '양수면 수출 우위(무역흑자), 음수면 수입 초과(무역적자)',
    'GDP_ANNUAL_GROWTH_RATE': '높을수록 경기 확장. 마이너스면 경기 침체 신호',
    'GDP': '국가 경제 규모의 절대적 크기. 전기 대비 증감으로 성장 판단',
    // ... 32개 전체
};
```

### 2. HTML 변경 (index.html 지표 헤더)

기존 지표 헤더(626~633라인) 아래에 해설 텍스트 추가:

```html
<p class="text-xs text-gray-400 mt-1"
   x-text="getGlobalIndicatorDescription(globalData.selectedIndicator)"></p>
```

## 32개 지표 해설 정의

| 카테고리 | 지표 | 해설 |
|----------|------|------|
| **무역/GDP** | 무역수지 | 양수면 수출 우위(무역흑자), 음수면 수입 초과(무역적자) |
| | GDP 성장률 | 높을수록 경기 확장. 마이너스면 경기 침체 신호 |
| | GDP (총생산) | 국가 경제 규모의 절대적 크기. 전기 대비 증감으로 성장 판단 |
| | GDP 농업 | 농업 부문 생산. 개도국일수록 비중이 높음 |
| | GDP 건설업 | 건설 경기 활성화 지표. 하락 시 투자 위축 신호 |
| | GDP 제조업 | 제조업 부문 생산. 산업국 핵심 성장 동력 |
| | GDP 광업 | 자원 수출국의 핵심 지표. 원자재 가격과 연동 |
| | GDP 공공행정 | 정부 지출 규모. 재정정책 확장/긴축 판단 |
| | GDP 서비스업 | 선진국 GDP의 60~80% 차지. 내수 경기 바로미터 |
| | GDP 운송업 | 물류/교역 활동 수준. 경기 선행 지표 성격 |
| | GDP 유틸리티 | 전기/가스/수도. 안정적이나 산업활동 변동 반영 |
| **고용** | 실업률 | 낮을수록 노동시장 호조. 과도하게 낮으면 임금 인플레이션 우려 |
| | 임금 상승률 | 높으면 소비 여력 확대, 동시에 기업 비용 증가·인플레 압력 |
| | 고용 변화 | 양수면 일자리 증가. 경기 회복/확장기에 증가 |
| **물가/소비** | 생산자 물가 지수 | 기업 투입 비용. 상승 시 소비자 물가 전이 가능성 |
| | 소비자 물가 지수 | 높으면 인플레이션 압력. 중앙은행 금리 인상 근거 |
| | 핵심 소비자 물가 | 식품·에너지 제외. 기조적 인플레이션 추세 파악용 |
| | 월간 소매 판매 | 양수면 소비 확대. 내수 경기 체감 지표 |
| **금리/금융** | 기준 금리 | 높으면 긴축(경기 과열 억제), 낮으면 완화(경기 부양) |
| | 중앙 은행 대차 대조표 | 증가면 양적완화(유동성 공급), 감소면 긴축(유동성 회수) |
| | 황금 보유 | 외환보유 안전자산. 증가 시 불확실성 대비 강화 신호 |
| **신뢰/선행** | 경기선행지수 | 100 이상이면 향후 경기 확장 예상, 미만이면 둔화 예상 |
| | 소비자 신뢰 지수 | 높으면 소비 심리 낙관. 소비 지출 확대 기대 |
| | 기업 신뢰 지수 | 높으면 기업 투자·고용 확대 의향. 경기 확장 신호 |
| **PMI/산업** | 제조업 PMI | 50 이상이면 제조업 확장, 미만이면 위축 |
| | 서비스업 PMI | 50 이상이면 서비스업 확장, 미만이면 위축 |
| | 산업 생산 | 양수면 산업활동 증가. 제조업 경기 직접 반영 |
| **재정/국가** | 정부 부채 | GDP 대비 비율로 판단. 높을수록 재정 건전성 우려 |
| | 재정 수입 | 세수 규모. 경기 호황 시 증가, 침체 시 감소 |
| | 재정 지출 | 높으면 재정정책 확장. GDP 대비 비율로 판단 |
| | 경상 수지 | 양수면 해외 소득 초과(흑자), 음수면 적자 |
| | 군사비 지출 | GDP 대비 비율로 판단. 지정학적 리스크 간접 지표 |
