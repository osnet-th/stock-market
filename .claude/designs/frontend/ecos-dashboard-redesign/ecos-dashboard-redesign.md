# 국내 경제지표 대시보드 리디자인

## 배경

현재 ECOS 페이지는 카테고리를 하나씩 선택해서 스프레드 카드와 원시 데이터 테이블을 보는 구조.
경제 전체 흐름을 한눈에 파악하기 어렵고, 숫자 위주라 상태 판단에 시간이 걸림.

## 핵심 결정

### 1. 미적 방향: 라이트 금융 대시보드

- **배경**: 기존 라이트 테마 유지 (bg-gray-50), 헤더/사이드바와 일관성 유지
- **숫자**: JetBrains Mono (CDN) 모노스페이스
- **본문**: Pretendard (한글 최적)
- **컬러 신호**: 녹색(안정) → 노랑(주의) → 빨강(위험) → 파랑(보수적)
- **카드**: 좌측 상태 컬러 바 + slate-50~white 그라데이션 배경으로 깊이감
- **게이지 바**: ref 레벨이 있는 지표에 시각적 위치 표시

### 3. 경제 펄스 (상단 핵심 지표)

모든 카테고리 진입 시 항상 표시. 선택된 카테고리와 무관하게 핵심 경제 방향성 요약:
- 기준금리, 장단기 금리차, KOSPI, 경기 방향성, CPI, 원/달러 환율
- 이들은 별도 API 호출 없이, 각 카테고리 데이터가 로드될 때 캐싱해서 표시
- (단, 현재 API 구조상 선택된 카테고리의 데이터만 가져오므로, 초기 로딩 시 INTEREST_RATE 데이터만 있음. 추후 확장 가능하지만 현재는 선택된 카테고리의 핵심 지표만 표시)

→ **수정**: 현 API 구조를 유지하되, 선택된 카테고리의 스프레드 중 첫 3~4개를 "하이라이트"로 상단에 크게 표시하는 방식으로 변경

### 4. 스프레드 카드 리디자인

각 카드에 다음 요소 포함:
- 지표명 + 툴팁(기존 유지)
- **값 (크게, 모노스페이스)**
- **미니 게이지 바**: ref 레벨이 있는 경우, 값의 위치를 시각적 바로 표현
- **상태 뱃지**: 안정/정상/주의/위험 (색상 코딩)
- 산식 (작게)
- 설명 텍스트 (작게)

게이지 바 예시 (ref 레벨이 있는 통화/금융 지표용):
```
보수적 ──────|── 정상 ──────|── 과잉
         ▲ 현재값(2.8)
```

### 5. 원시 지표 테이블

기존 라이트 테마 유지:
- 행 호버 시 미묘한 하이라이트 (기존 유지)
- 변화율에 색상 적용 (상승: 빨강, 하락: 파랑 - 한국 주식 관행)
- 모노스페이스 숫자로 가독성 향상

## 기술 구현

### 변경 파일

1. **`index.html`** - ECOS 섹션 (line 388~923) HTML 구조 변경
2. **`css/custom.css`** - 다크 테마 스타일, 게이지 바 애니메이션 추가
3. **`js/components/ecos.js`** - 하이라이트 지표 추출 헬퍼 추가

### 변경하지 않는 것

- API 구조 (백엔드 변경 없음)
- 다른 페이지의 디자인 (ECOS 페이지만 다크 테마)
- 기존 스프레드 계산 로직

### CDN 추가

```html
<!-- JetBrains Mono (숫자 전용) -->
<link href="https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;600;700&display=swap" rel="stylesheet">
<!-- Pretendard (한글 본문) -->
<link href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable-dynamic-subset.min.css" rel="stylesheet">
```
