# DART 재무정보 UI 개선 - Brainstorm

**Date:** 2026-03-16
**Status:** Decided

## What We're Building

포트폴리오 국내 주식의 DART 재무정보 표시를 "요약 대시보드 + 개선된 상세 테이블" 구조로 개선한다. 기업 분석 시 핵심 지표를 빠르게 파악하고, 상세 데이터도 편리하게 확인할 수 있도록 한다.

### 현재 상태

- 8개 메뉴(재무계정, 재무지표, 전체재무제표, 주식수량, 배당정보, 소송현황, 사모/공모자금)
- API 응답의 키를 동적으로 컬럼화하여 raw 테이블로 표시
- 숫자 포맷팅, 증감 색상, 높이 제한 없음
- 데이터가 많으면 스크롤이 길어지고 핵심 정보를 찾기 어려움

### 변경 범위

1. **요약 대시보드**: 각 메뉴 상단에 핵심 지표를 카드형으로 배치 (전년 대비 증감 색상)
2. **테이블 개선**: 숫자 포맷팅, 증감 색상, 고정 헤더, 컴팩트한 높이 제한

### 변경하지 않는 것

- 8개 메뉴 구조 (기존 유지)
- 백엔드 API (프론트엔드만 변경)
- 필터 옵션 (연도, 보고서, 지표분류 등 기존 유지)

## Why This Approach

- **요약 + 상세 이중 구조**: 핵심 정보 빠른 파악 + 깊이 있는 분석 모두 지원
- **기존 테이블 유지**: 동적 구조를 버리지 않고 개선하여 모든 메뉴에 범용 적용 가능
- **프론트엔드만 변경**: 백엔드 변경 없이 UI만 개선

## Key Decisions

| 결정 사항 | 선택 | 이유 |
|----------|------|------|
| UI 구조 | 요약 대시보드 + 상세 테이블 | 핵심 파악 + 상세 분석 모두 지원 |
| 요약 카드 | 각 메뉴별 핵심 지표 카드 | 전년 대비 증감 색상으로 직관적 |
| 테이블 개선 | 포맷팅 + 색상 + 높이 제한 + 고정 헤더 | 가독성 개선 |
| 차트 | 이번 범위에서 제외 | YAGNI, 요약 카드로 충분 |

## 요약 카드 설계 방향

각 메뉴별로 핵심 지표 3~5개를 카드로 표시:

### 재무계정 (accounts)
- 자산총계, 부채총계, 자본총계, 매출액, 영업이익, 당기순이익

### 재무지표 (indices)
- 선택된 지표분류의 상위 지표들 (ROE, ROA, 부채비율 등)

### 배당정보 (dividends)
- 주당배당금, 배당수익률, 배당성향

### 기타 메뉴
- 해당 메뉴의 핵심 1~3개 항목을 카드로 표시

## 테이블 개선 방향

- 숫자 컬럼: 천단위 구분, 우측 정렬
- 증감 표시: 전년 대비 증가(빨강)/감소(파랑) 색상
- 높이 제한: max-height + overflow-y scroll
- 고정 헤더: sticky header
- 행 호버: hover:bg-gray-50

## Technical Notes

- 프론트엔드만 변경 (index.html, app.js)
- Chart.js는 이번 범위에서 사용하지 않음
- 요약 카드 데이터는 financialResult에서 추출 (추가 API 호출 불필요)
- 동적 테이블 구조 유지 (Object.keys 기반)
