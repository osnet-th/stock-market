# 글로벌 경제지표 추가 (Issue #25)

## 배경

기존 55개 글로벌 경제지표에 10개 신규 지표 추가, 1개 제거, 신규 카테고리 1개 추가.

## 변경 요약

| 구분 | 내용 |
|------|------|
| 신규 카테고리 | 통화량 (MONEY_SUPPLY) |
| 신규 지표 | 10개 |
| 제거 지표 | 1개 (군사비 지출) |
| 최종 지표 수 | 55 - 1 + 10 = **64개** |

## 상세 변경 내역

### 1. 신규 카테고리: 통화량 (MONEY_SUPPLY)

| enum | pathSegment | displayName |
|------|------------|-------------|
| MONEY_SUPPLY_M0 | money-supply-m0 | 1개월 미만 통화 공급량 |
| MONEY_SUPPLY_M1 | money-supply-m1 | 1개월 통화공급량 |
| MONEY_SUPPLY_M2 | money-supply-m2 | 2개월 통화 공급량 |
| MONEY_SUPPLY_M3 | money-supply-m3 | 3개월 통화공급량 |

### 2. 금리/금융 (RATE_FINANCE) 추가

| enum | pathSegment | displayName |
|------|------------|-------------|
| LENDING_RATE | lending-rate | 대출금리 |
| INTERBANK_RATE | interbank-rate | 은행간 금리 |

### 3. 재정/국가 (FISCAL_STATE) 추가

| enum | pathSegment | displayName |
|------|------------|-------------|
| FOREIGN_EXCHANGE_RESERVES | foreign-exchange-reserves | 외환 보유고 |

### 4. 물가/소비 (PRICE_CONSUMPTION) 추가

| enum | pathSegment | displayName |
|------|------------|-------------|
| INFLATION_RATE | inflation-rate | 물가상승률 |
| INFLATION_EXPECTATIONS | inflation-expectations | 인플레이션 기대 예상치 |
| INFLATION_RATE_MOM | inflation-rate-mom | 인플레이션(월간) |

### 5. 제거: 군사비 지출

- `MILITARY_EXPENDITURE` enum 제거
- 메타데이터, 프론트엔드 설명 제거

## 수정 파일 목록

### 백엔드

| # | 파일 | 변경 내용 |
|---|------|----------|
| 1 | `IndicatorCategory.java` | `MONEY_SUPPLY("통화량")` enum 추가 |
| 2 | `GlobalEconomicIndicatorType.java` | 10개 enum 추가, `MILITARY_EXPENDITURE` 제거 |
| 3 | `global-indicator-metadata.yml` | 10개 메타데이터 추가, 군사비 지출 제거 |

### 프론트엔드

| # | 파일 | 변경 내용 |
|---|------|----------|
| 4 | `global.js` | 10개 지표 설명 추가, 군사비 지출 설명 제거 |

## 주의사항

- `GlobalCategoryResponse.fromAll()`은 enum을 동적으로 읽으므로 API 자동 반영
- 배치 스케줄러는 `GlobalEconomicIndicatorType.values()`를 순회하므로 자동 반영
- DB의 기존 `MILITARY_EXPENDITURE` history/latest 데이터는 유지 (삭제하지 않음)
- 최종 지표 64개 x 1.5초 = 약 96초 배치 소요 예상
