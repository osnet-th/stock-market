# 경제지표 단위 변환 통합 표시

## 배경

현재 경제지표 값과 단위가 분리되어 표시됨 (예: `2,148,225.3` + `십억원`).
사용자가 직관적으로 이해하기 어려우므로, 단위를 계산하여 하나로 합쳐서 표시한다.

## 핵심 결정

- **변환 로직은 프론트엔드(format.js)에서 처리** — 백엔드 변경 없음
- 변환 가능한 단위만 변환하고, 나머지는 `값 + 단위` 그대로 붙여서 표시
- 카드뷰와 테이블뷰 모두 통합 표시 적용

## 단위 변환 규칙

| unitName | 변환 방식 | 예시 |
|---|---|---|
| `십억원` | ÷10,000 → 조원, 그 외 → 십억원 유지 | 2,148,225.3 → `2,148.2조원` |
| `백만원` | ÷1,000,000 → 조원, ÷10,000 → 억원 | 76,480,638 → `76.5조원` |
| `백만달러` | ÷10,000 → 조달러, ÷100 → 억달러 | 13,259.1 → `132.6억달러` |
| `천달러` | ÷1,000,000 → 조달러, ÷100,000 → 억달러 | 427,624,953 → `4,276.2억달러` |
| `천명` | ÷10,000 → 억명, ÷10 → 만명 | 29,406 → `2,940.6만명` |
| `천㎡` | 그대로 `값 천㎡` | 7,165.11 → `7,165.1천㎡` |
| `%` | 값 뒤에 % 붙임 | 2.5 → `2.5%` |
| `원` | 값 + 원 | 1,500.6 → `1,500.6원` |
| `달러` | 값 + 달러 | 36,855.1 → `36,855.1달러` |
| `달러/배럴` | 값 + 달러/배럴 | 68.51 → `68.51달러/배럴` |
| `달러/트로이온스` | 값 + 달러/트로이온스 | 5,014.65 → `5,014.65달러/트로이온스` |
| `조원` | 그대로 `값조원` | 12.9 → `12.9조원` |
| 지수류 (`2020=100` 등) | 값만 표시 (단위는 tooltip 등에서 확인) | 5,781.2 |

## 변환 임계값 (한국식 단위)

- **원화 계열**: 1조 이상 → 조원, 1억 이상 → 억원, 그 외 원래 단위 유지
- **달러 계열**: 1조 이상 → 조달러, 1억 이상 → 억달러, 그 외 유지
- **명 계열**: 1억 이상 → 억명, 1만 이상 → 만명
