GitHub 이슈를 조회하고 내용을 출력해줘.

## 1. 이슈 조회
- `$ARGUMENTS`가 이슈 번호(숫자)인 경우, 해당 이슈를 바로 조회해.
- `$ARGUMENTS`가 비어있거나 숫자가 아닌 경우, `gh issue list --state open --limit 20`으로 열린 이슈 목록을 가져와.

## 2. 이슈 선택 (목록 조회 시)
- 이슈 목록을 번호, 제목, 라벨과 함께 보기 좋게 정리해서 보여줘.
- AskUserQuestion으로 조회할 이슈 번호를 선택하게 해.

## 3. 이슈 상세 조회
- `gh issue view {이슈번호}`로 이슈 상세 내용을 가져와.
- 다음 정보를 정리해서 출력해:
  - 제목
  - 상태 (open/closed)
  - 라벨
  - 작성자
  - 작성일
  - 본문 내용
  - 댓글 (있는 경우)

## 4. 후속 작업 안내
- 이슈 내용 출력 후, 다음 가능한 작업을 안내해:
  - 이 이슈를 기반으로 브랜치 생성 및 작업 시작
  - 다른 이슈 조회
