# 국내 경제지표 툴팁 → 클릭 상단 설명 변경

## 배경

국내 경제지표의 물음표(?) 아이콘이 현재 호버 툴팁으로 동작하지만, 재무제표처럼 클릭 시 상단에 설명 박스가 나타나도록 변경.

## 변경 대상

1. **스프레드 카드** (`index.html:458-467`) - 호버 툴팁 → 클릭 버튼
2. **원시 지표 테이블** (`index.html:621-632`) - 호버 툴팁 → 클릭 버튼

## 구현 방식

재무제표의 `portfolio._tooltipText` 패턴을 그대로 적용:

1. `ecos` 상태에 `_tooltipText: ''` 필드 추가 (`ecos.js`)
2. 스프레드 카드/원시 지표 테이블 상단에 설명 박스 HTML 추가 (`index.html`)
3. 기존 호버 툴팁을 클릭 버튼으로 교체 (`index.html`)
